HTML Elements

An HTML element is defined by a start tag, some content, and an end tag.
Don't worry if we use tags you have not learned about yet.

HTML Elements

The HTML element is everything from the start tag to the end tag: <html> The HTML elements can be nested (this means that elements can contain other elements). All HTML documents consist of nested HTML elements. The following example contains four HTML elements<html,body,h1> and </p>.
