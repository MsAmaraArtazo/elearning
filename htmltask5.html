HTML Formatting

HTML contains several elements for defining text with a special meaning. Formatting elements were designed to display special types of text: <b>Bold Text,</strong>Important Text </i>Italic Text,</em>Empasized Text,</mark>Mark Text </small>Smaller Text,</del>Deleted Text,</ins>Inserted Text </sub>Inserted text and</sup>Superscript Text.
